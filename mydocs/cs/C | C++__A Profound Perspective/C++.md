## 新引入的数据类型

### 布尔数据类型

布尔-bool 类型，用来表示 `true` 或 `false`，占一个字节。

bool 型转 int 型，`true` 转化为 1，`false` 转化为 0。

int 型转 bool 型，0 转化为 `false`，任何非零整数转化为 `true`。
